# 🔐 Awesome Cyber Security

Coleção de ferramentas, scripts, recursos e materiais educativos sobre **Cyber Segurança**, totalmente éticos e focados em aprendizado.

## 📁 Conteúdo
- 🛠 Ferramentas criadas em Python
- 📘 Listas de estudo e materiais
- 🎯 Scripts simples e seguros para pentest educativo
- 🔍 OSINT
- 🧪 Testes básicos de vulnerabilidade

## 🧰 Ferramentas incluídas
| Ferramenta | Descrição |
|-----------|-----------|
| Port Scanner | Scanner de portas simples (TCP) |
| OSINT Email Checker | Verifica vazamentos públicos de e-mail (API simulada) |
| File Integrity Monitor | Monitora alterações em arquivos |
| Log Analyzer | Identifica padrões suspeitos em logs |
(algo básico, apenas para não ter um perfil vazio).
